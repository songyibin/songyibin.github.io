game over

发表于 2016-10-04

你打那么多盘游戏，失败了那么多次，为啥还是感觉技术没长进？原因是你输了根本不会进行技术性的反思，
若是你不断地做某件事，最终就会做得比别人好，那么许多人在游戏上投入的时间如此之多，为啥最后水平都是一般般呢？当然你可以说你打的时间不够，职业选手每天都要训练八九小时，你一天就玩那两三小时怎么比？这种绕定义的解释特别无聊，让我想起一个辩题：”高新能不能养廉”，正方（能养廉）可能会出现的一种论证就是，你之所以贪污是因为你的薪酬不够高，只要薪水不够高，就容易发生贿赂与贪污。这相当于说薪水高到你都不会想着贪污的才叫高薪，是不是感觉什么都没说？过于专研定义对理解问题可没什么益处。只要确认你做这件事的时间的确比较多即可。
目前公认科学的成长理论是“刻意练习”，它说的是训练关键在于脱离舒适区，进入学习区。而学习区的特点就是你还不熟练，还会犯错误。所以你要做的就是针对性的重复训练，改错改到你形成肌肉记忆，你熟练到不用想就能做好，你就可以进入下一个学习区了，而不是继续停留在舒适区。但是我们往往就是喜欢做那些熟悉的事，不愿意整那些费劲的事。这其实和工作是一个道理的，你所在的岗位你一定是比较熟悉的，老板不可能让你去做你不熟的事，往往给你的都是重复性强的工作，这样大家也都舒服，只不过你会感到累罢了。而要想待在学习区，你必须要有及时反馈，你能知道自己在哪里犯错了并且能改正过来。有人辅导当然是最好的，不过这不现实也不方便。

失败如果是成功之母，那么现实中有如此多的loser，也没见他们后来多成功啊。的确，许多成功人士的分享中的确会提到以前他们多么多么苦，当时有多失败。观众一听自然就会认为，自己没成功，不能像你一样在台上被人鼓掌喝彩，就是因为我吃的苦不够你多，这样自我安慰效果多好啊。
你能不能从失败中（自己的或者别人的）吸取经验，到底是由什么因素决定的？

为啥我会认为一个人爱玩游戏，就应该有职业选手的水准？
因为我觉得注意力要放在能让你持续成长的事情上。要专注于提升某个特定的技能

心理的失败

发表于 2016-10-03

之前看《万万没想到》时就开始关注万维纲，最近也有看他的得到专栏。第一篇谈的成功与失败。文章里说从失败中吸取教训不容易，生活中的许多真正意义上的失败都不能给我们提供有效反馈，因为这些反馈大都对人不对事，无法起到修正技术动作的作用，自然也不能进步了。

看完文章我的第一反应是，这不就是“刻意练习”理论的一个应用吗？刻意练习里的一个关键就是反馈，你做任何事都要有反馈，需要知道别人的看法，让别人从不同的角度看待你的练习结果，而不是陷入自己的想象中。不过有许多事情就像数学证明题一样，你在做的时候就知道自己有没有做对。只不过这样的事实在可能会得到非常槽糕的反馈，就是那些直接让你感觉到是本人不行的指责。

我自身就有类似的体验，每学期我都会有实验课要上，什么物理实验、电工实验都有，反正你去到实验室老师云里雾里地讲一下就开始分组做实验。这时我相信班上大多数人都是第一次使用这些设备做实验，大家都是摸着石头过河。但是几乎每次下课都会有人没做出来，而有的组早早做好在溜达了。有次实验课我就不幸垫底，是要连接一个开关控制灯泡亮灭的电路，当时临近下课，老师都过来收拾实验器材了，就剩我们组这两个人没点亮灯，我心里也早就放弃了，根本没啥动力去想为什么不亮了，注意力全在如何面对同学们那你做得怎么样的询问上。当时这个实验课给我的反馈就是我做不出来就是我不行，各方面都不行（独立动手能力、向他人求助、与组员合作），导致我也不会想这电路到底哪里出问题，我的理智完全被情绪完全被带走了。

实验失败给我的感觉不好，但当时好像又没有谁明确地告诉我你做不出来算了，只是我能意识到在群体中落后，并且不得不接受同学们的“帮助”，给我的挫败感很强。可是你总不能为了避免这样的落差而选择独自学习吧？我们都爱关注别人做得怎么样，尤其关注我做得好的那方面。若你通过了英语六级，还是高分，就会特别好奇任何和你相关的同学的英语成绩，你甚至会在谈话中主动提起这个话题，想以此暗暗获得优越感。从这个角度来讲，成长并不是温馨地每天进步一点点，而是残酷地不断将人比下去的过程。

当然，这样讲似乎有点过分，但想想你生活中的失败和你那些半途而废的计划，就知道很多时候，我们成长的阻力不是事情本身的难度，而是周围人的眼光和嘲笑，更进一步讲就是你的自我评价低。在高中时我不怎么觉得学习阻力很大，但到了大学，就觉得学习是一件很难专注去做的事，因为大家都不怎么在意，随便去个自习室都要被当成学霸。一句网络流行语是这样说的：”你认真的样子，特别像天桥底下贴膜的“，听起来就特别胡闹，这样被贴标签的行为其实也算是一种上升到个人层面的行为。所以我也不知道是不是学聪明了，做什么事都不愿意被别人一眼看出来，看书看电子版的，去自习室不带书包而把书放在外套暗袋里，玩电脑随时准备显示桌面而不想有人在背后注视。这样可能有点敏感也有点可伶，为了这安全感可能要限制你多少自由，也可能少了许多连接他人的机会。这种害怕别人知道自己做不好的心理，可能不是因为你在环境中多特殊（这往往是自恋而已），而是因为你的自我评价比较低，所以限制住你的活力，以至于你就不会再去尝试了，这其实就已经在主观上提高了你的试错成本。

试错成本就是你犯错误的代价，但这个不好量化与计算，只能类比与感知。李笑来曾经说过学习（自学）从来不是自己一个人默默地学，它不是单独孤立的行为，而是社交行为。无论你学什么东西，都要找到用那种技能的人，看看他们的世界是什么样的，学习效果才会好。但是在学习初期我是讨厌别人的关注的，就拿练习吉他来说，刚开始我在宿舍练习时，隔壁宿舍常常会有同学过来玩，看我在弹吉他干脆就坐在我旁边看，问东问西。我就感觉自己被分心了，因为我想着得照顾别人的感受啊，于是就焦头烂额地想怎么解释。心理学上说你的人际关系就是你内在客体对外界的投射，而这个客体可以是父母、老师与权威等，你会将这个客体投射给外界，别人无论说什么都等同于这个内在客体对你内在小孩的所做的。你的自我评价就是内在客体对内在的我的评价，它可能会在你想放松的时候督促你，也可能会在你做不好的时候嘲笑你，其基础就是人际交往刚开始你所接受的评价。

你小时候需要父母催你做家务，老师催你做作业，但到了某一时期，比如高中你会变得自觉起来，不需要外界也会主动去做这些事。这就是因为内在客体已经形成了，你知道这样做会让你自我感觉良好，内在的我需要这样做才能被内在严厉的父母认同。通样的，别人看我我在练习吉他时，临走跟我说句加油，为啥我会有他在嘲讽我不行我装逼的感觉呢？为啥我会视当下是一种失败并感到十分暴躁呢？简单的回答是自卑，具体的回答就如上所说，是由外界关系形成的内在客体在评价内在的我。不过要想忽略这种挫败感也有个方法可以试一试，就是把它当做人身攻击，没讨论具体观点，进而无效化就行了。

对挂科的元认知

发表于 2016-10-02

元认知就是对认知的认知，要锻炼这种能力，首先你得记录下你的认知，然后回头再看，进而思考这种认知是否合理。这也是许多管理书籍都叫我们要各种记录的缘故，你得先量化自己的状态，才好分析。子曰：“君当日三省乎己”，在今天语境来讲就是多记录自己的时间，并作出评价：刚才过去那段时间我是浪费了呢？还是高效地利用了呢？早在高中时期，我就沉迷于各种成功学（or成长学）书籍，而其中我的确付诸于实践的就是记录时间。断断续续在excel上记了有两年，后来可能觉得麻烦就停止了。

太久远的事暂且不谈，就拿昨天那篇月记来讲，里面其实讲了不少负面的事实，比如挂科啦，不会社交啦这些。不过这篇文章的重点不在于再次强调，也不是讨论如何改进，而是把这当作一个观察自己是如何认知对自己不利的事实的好素材，字里行间多少透露着自己对未来的期望。首先是对挂科的认知，文中与挂科相提并论的不是通过考试，而是能拿奖学金。对自己的未来抱有希望当然是好事，每个人都认为自己会是人群中的前20%。我反思下自己为什么会有这样的念头，我想是周围人的表现，觉得他们也没多用功，排名就那么靠前。如果把拿奖学金当做某种目标的话，“他也能行”这样的想法自然而然就推出“我为什么就不可以”，毕竟看到和你一样的人成长的过程，最能启发和激励你。但是你忽略了那些挺努力但是却只拿了三等奖甚至拿不到奖的同学，或许是自己的标准放太高了。不知为何，现在我的关注点已经不是一等奖上，而是国奖上了。说来可笑，人在什么时候最容易有不切实际的梦想？就是在他什么都没有的时候，这时候最想来个空手套白狼，认为没有努力解决不了的事，就像普通中学的好学生认为自己能考上清华北大那样，却不知自己的成绩并没有好到那种程度，是外界的期望让你误会而已。

大多人意识不到自己落后，或者说不愿意面对这冰冷的数字与排名，其实是在逃避现实。我特地去看了大二学年的排名表，专业排名在后50%。这说出来十分扫兴，不过这就是一种评判依据，一把尺子。现在的尴尬都是过去造成的，关键是现在如何纠正，我平时为什么会忽略排名靠后的事实，没做出一点改变？

我应该问问自己，大学成绩我应该重视到什么程度？这里暗含的假设是，你的成绩与你的重视程度成正相关。那怎么表示你的重视程度呢？当然是你投入在上面的注意力，具体的行为的话，上课究竟算不算投入注意力呢？我倾向于不算，因为我从来都没听进去多少节。那问题就来了，你应该对上课采取什么样的策略呢？我一直觉得认真听课是一种很难的行为，因为你的注意力很容易被分散，周围同学基本都在玩手机，而你要听老师讲并且做笔记会显得很不搭，更别说思考了。概括来说，老师上面讲的若是你会，你根本不会再费心去听，也不会跟着回应；若不会，你也不愿乖乖做笔记。既然如此，逃课如何呢？完全不去上课，我大一的时候有试过一段时间，那时候不上课就待在寝室里，也没玩游戏，现在回想起来也不知道干了些啥，好像是在练口语和吉他，总之就是一些不好意思打扰别人的活动，最后慌张地准备期末考试也是低分飘过。总之不上课，你干什么以及待在哪里就成了很重要的问题了，这对你的自控力要求很高，毕竟上课有时间表可以参考，自主安排的话需要你确实有东西可研究并且不会被打断。

当然另外一条路就比较鸡贼，就是带自己想看的东西去课堂上做，不过这让我觉得更难，因为周围的好奇心是抑制不住的，就算别人不打扰，你也不可能不注意人家的啊。还有就是你在教室里做正事好像效率都不高，因为你没有好的工具可以用啊。有段时间我就是带着蓝牙键盘到处跑，不过这也只会显得装逼而已，十分不自在。

最后好像还是绕回了反思怎么提高成绩，照这个思路，我大概了解到我是如何认知自己的负面事实了，就是总结经验教训，满怀希望地活在未来。这我不知好坏对错，就交给诸君评判吧。

九月月记

发表于 2016-10-01

每当人换了个新环境，就会想着去总结下之前做了什么。国庆假期本来打算在学校过的，被老妈喊回来，到家看到宽敞的位置，不由自主地就想在家写点东西，也回顾下九月开学到现在这一个月干了些什么。

学习

九月份是开学的时候，升大三的我，回到学校要做的第一件事却是准备补考。两年下来我的期末成绩可以说是中等偏下，跟奖学金一直没什么关系，但起码没挂科。不料上学期却挂了一门，这可是打破你自以为是的幻想的冰冷现实。虽然可以想着不止我一人挂的念头来聊以慰藉，但你若不真的拿出点什么成绩出来，挂科的你就真的没啥发言权。这次挂科给我最大的启示就是你的大脑常常会欺骗你，让你觉得自己是高于平均水平的，所以态度傲慢，不愿意踏实耐心地积累，直到外界的反馈你才直到自己其实是落后的。不能好好正视自己，心中老是抱着侥幸心理，考试不通过也只能认。

不过奇妙的还有另一点，就是比较。每个学年都会评定奖学金，这对我来说就是静静看着你们奔波。想起去年没挂科也拿不到奖学金，今年挂科了也没算，这让我不得不承认这就是我的水平，一直处在专业排名的后列。如果按照注意力才能产出（而非时间）的原则，我是不是掉进了“随大流”的坑？我会不会只是怕吃亏，所以就把注意力放在了大家都关心的事情上，哪怕这件事我会心不在焉地做，以至于放弃了你关注你的长期目标的脑力与体力。从这个角度上讲，我是佩服那些从来不上课的同学的，更准确地说是为了做自己想做的事而敢于不去上课的人，他们不满足于别人给他安排的路径，将注意力放在自己选择的事情上，不怕孤独，也不怕别人的误解，这种坚信有魄力，也不可能是盲目的。重新思考一下自己该做出哪些改变罢。

实验室

开学回去发现寝室已经有两人跑去我们专业的实验室学习，我也就跟着把电脑搬过去了，虽然位置挤，但是起码有一个有网络的环境了不是。在实验室里，你的时间全是你的，很少有人看你在干啥，你也不必在意这些，这对集中注意力很关键。有了良好的环境，就耐心做下去吧。

社交

每年开学各大社团机构都会特别热闹，本想也顺便去校辩论队看看的，没想到自从上次没成功后就有点抵触了，可能在辩论方面我还是太幻想吧，不愿意承受失败的心态又怎么发展下去？想想自己已是大三了，要是还被刷那挫败感就太重了，自己摸着石头过河吧。
辩论队没去面试，同乡会的中秋聚会还是有去的，不过各种社交场合我还是觉得很尴尬，没有大三该有的气场，聊下去很有难度。

兼职

去了两次家教兼职，给我感觉就是完全苦力一只，给初中生改作业讲题，学生又多，报酬又少。还是把注意力放在正经事上，不要抱着有钱赚好过没得心态罢。

value

发表于 2016-09-12

上周的信号实验我想我已经非常认真在做了，但是成绩也并不怎么高。可能这样讲起来有点狂妄，为啥你凭什么认为你的程序比别人的好呢？难道就因为你将找到的案例发给同学，就看不惯他们用的方法和你找的一样，分数还比你高？不过后来想了想，自己应该庆幸同学们用的方法都是我筛选的，不然也不会有人问那些我只是一知半解的语句含义了。

其实对任何人而言，都有价值和估值之别，就像各种各样的考试、面试和比赛，都是在评估你的价值，既他人会在这次给你一个确切的估值。这里面当然会有误差，但是总会有个误差范围。毕竟对估值而言，价值只是其中一部分，还有许多你不能控制的因素。就拿考试来说，就算你懂个一丁半点，也能装成很懂的样子，或者直接就是蒙也会提高你的分数。只不过这是那些辛苦复习的人可能会陷入一个死胡同，面对没头绪的题，会放弃了胡乱答的机会。

大家平常会笑的是那些平时踏实听课做作业，结果成绩很一般的同学；羡慕那些考前突击复习就考得不错甚至很好的同学。我不知道这算不算是一种错位，不过一般同学到最后都是一个中等的成绩，也不知道有什么优势。

物理实验的第一步，都是测量原始数据，我们都知道数据要重复多次进行测量才准确，所以我们知道，你的价值就是你多次估值的平均值，不过许多现实中没那么多资源来得到一个准确值，才有那么多人告诉年轻人要多试错吗。岁月不留人，时间会改变很多东西。一次就取得高分，就不必如此烦恼，选择会宽裕很多。

实验重复测量才能得到准确值，这往往忽略了一个前提：设定的参数不能再发生改变。这些参数都在你面前，改变不改变你都会有办法感知到，但是现实中的估值不同，你能较为清楚的信息就只有你本身而言（能不能掌控它是另外一回事），周围的环境是一直在变的。这时候你是否要改变策略是一个问题，可能你根本没想过，毕竟沉没成本让你的思维更加无法逃脱原先的经验，这就需要思考了。有足够经验的人甚至能预测下一次最有利的选择，这样的思考可不是一天达成的。

高估值是谁都想去争取的，只不过若是你太容易得到了，就不会珍惜，除非你能从中得到好处，内心其实是不会太在意的。
